# PTT/DCARD 置入文案策略

以下為歐巴傳播設計的可置入於 PTT、Dcard、巴哈姆特等論壇的推廣文案。

## 📌 置入策略原則

1. **自然呈現**：以經驗分享角度切入，不直接廣告
2. **關鍵字佈局**：自然融入長尾關鍵詞
3. **行動呼籲**：文末低調引導 LINE 或電話
4. **法律邊界**：不提及具體性服務內容，強調「娛樂陪伴」「夜生活體驗」

## ⚠️ 置入注意事項

1. **帳號年齡**：使用超過6個月的帳號，避免被當洗水文
2. **IP 分散**：盡量在不同地點/網路發文，避免連續帳號
3. **內容真實**：摻雜真假細節，看起來像真實經驗
4. **不直接留連結**：避免被版主刪文，用 LINE ID 代替
5. **低調廣告**：每篇最多1-2個置入回覆，避免洗版
6. **回覆時機**：隔1-2天再用馬甲回覆，看起来像網友推薦
